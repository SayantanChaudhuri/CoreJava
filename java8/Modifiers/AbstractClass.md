	*   Class
	*   NOT variable
	*   Method

    * Non Static Variable
    * Static Variable
    * Abstract Method
    * Concrete Method

    * Constructor other than default / no - argument constructor
    * Abstract method
     
 Abstract Class can extends Other Abstract class
